QUICK START GUIDE
=================

For complete documentation with all examples, see: HOW_TO_USE.md

INSTALLATION:
-------------
1. Install python 3.9.xx - 3.14.xx, be sure to add python to PATH while installing it.

2. Install Microsoft Visual C++ Redistributable:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

3. Run: poetry install

4. Install Firefox add-on for cookies:
   https://addons.mozilla.org/en-US/firefox/addon/cookies-txt-one-click/

COOKIES SETUP:
--------------
Export cookies and save as default.txt in:
- HBO Max: fuckdl\cookies\HBOMax\default.txt
- Amazon: fuckdl\cookies\Amazon\default.txt
- Apple TV Plus: fuckdl\cookies\AppleTVPlus\default.txt
- iTunes: fuckdl\cookies\iTunes\default.txt
- Netflix: fuckdl\cookies\Netflix\default.txt
- And so on for other services...

CREDENTIALS:
------------
For DisneyPlus, Videoland, ParamountPlus, All4, RakutenTV, BritBox:
Add credentials in fuckdl\config\fuckdl.yml

QUICK EXAMPLES:
---------------
# Amazon Prime Video - Single episode
poetry run fuckdl dl -al en -sl en -w S01E01 Amazon https://www.primevideo.com/region/eu/detail/0KRGHGZCHKS920ZQGY5LBRF7MA/

# Amazon Prime Video - Entire season
poetry run fuckdl dl -al en -sl en -w S01 Amazon https://www.primevideo.com/region/eu/detail/0KRGHGZCHKS920ZQGY5LBRF7MA/

# iTunes - Download movie/show
poetry run fuckdl dl -al tr -sl tr iTunes -m umc.cmc.2lj6d47e7094s6ss83j0uppdm

# iTunes - 4K HDR
poetry run fuckdl dl -al en -sl en -v H265 -r HDR iTunes -m umc.cmc.2lj6d47e7094s6ss83j0uppdm

# Apple TV Plus
poetry run fuckdl dl -al en -sl en -w S01E01 AppleTVPlus https://tv.apple.com/us/show/big-beasts/umc.cmc.7d9yulmth1rvkwpij477qsqsk

HELP:
-----
# General help
poetry run fuckdl dl --help

# Service-specific help
poetry run fuckdl dl Amazon --help

CDM DEVICES:
------------
For SL2000/SL3000 devices:
1. Create folder in fuckdl\devices\
2. Add bgroupcert.dat and zgpriv.dat
3. Edit fuckdl.yml to add folder name
4. FuckDL will auto-create PRD files and reprovision hourly
