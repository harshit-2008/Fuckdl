DisneyPlus:

1) Use this command to list the available extras, then you select which extra you want to download:

poetry run fuckdl -al all DisneyPlus -extras list entity-23c8484e-a657-440b-8bf9-678f547331ff

2) Use this command to download all extras at once:

poetry run fuckdl -al all DisneyPlus -extras all entity-23c8484e-a657-440b-8bf9-678f547331ff

3) Use this command to list all the available versions, then you select which version you want (imax, etc):

poetry run fuckdl -al all DisneyPlus -versions list entity-ef8fe7d7-2ba1-4c95-acf3-27ab28df66e8

4) Use this command to download all the available versions at once:

poetry run fuckdl -al all DisneyPlus -versions all entity-ef8fe7d7-2ba1-4c95-acf3-27ab28df66e8
